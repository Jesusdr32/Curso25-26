Resuelve los ejercicios siguientes utilizando métodos del objeto Array, de forma que muestren los resultados por consola. No es necesario que escriban los resultados en el documento.

1. Dado el array [15, 3, 7, 22, 10], ordénalo de manera ascendente utilizando el método sort

2. Del array ['casa', 'árbol', 'sol', 'perro', 'gato'], filtra las palabras que tienen más de cuatro letras

3. Dado el array [25.99, 10.5, 5.99, 100, 1.5], utilizando el método map para crear un nuevo array donde cada precio se duplique

4. Del array [1, 7, 15, 22, 5, 10], filtra los números que son pares

5. Ordena alfabéticamente el array ['perro', 'gato', 'elefante', 'jirafa'] utilizando el método sort

6. Dado el array de fechas ['2023-05-10', '2022-12-01', '2024-02-20'], utiliza el método map para crear un nuevo array con objetos Date correspondientes a cada fecha

7. Del array [8, 15, 4, 20, 10], filtra los números que son mayores que 10

8. Dado el array de objetos [{nombre: 'Juan', edad: 25}, {nombre: 'Ana', edad: 30}, {nombre: 'Pedro', edad: 20}], ordénalos de menor a mayor según su edad.

9. Dado el array ['hola', 'adiós', 'buenos días'], utiliza el método map para crear un nuevo array donde cada palabra esté en mayúsculas

10. Del array de objetos que representa libros, filtra aquellos cuyo año de publicación sea posterior a 2000
